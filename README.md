# Beer store

使用 Vue CLI3 及 Vue Router 完成 SPA 互動式電商網站，涵蓋前後台，串接 API 實做出商品管理 (CRUD)、購物車、優惠券、登入/登出介面等功能，並搭配 ESLint Airbnb維持程式碼品質.

### 使用技術

- Vue Cli 3
- VueRouter
- ESlint 的 Airbnb 優化程式碼
- RWD 響應式網頁設計
- Bootstrap4
- SCSS
- TinyPNG (縮小圖檔)
- API串接 (由六角學院提供)
